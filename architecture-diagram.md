═══════════════════════════════════════════════════════════════════════════════
                              KEY TECHNOLOGIES
═══════════════════════════════════════════════════════════════════════════════

  • Framework:          .NET 10.0 (Blazor Server - Interactive Server Mode)
  • ORM:                Entity Framework Core 10.0
  • Database:           SQL Server (with InMemory option)
  • Authentication:     Google OAuth 2.0 + Cookie Authentication
  • Mapping:            AutoMapper 12.0
  • Image Processing:   SixLabors.ImageSharp 3.1
  • Architecture:       Feature-based organization
  • Security:           User Secrets, Antiforgery Protection
